# Data Persistence Project

Standalone Unity project for the Junior Programmer data persistence submission.

## What is implemented

- Main menu with player name entry.
- Persistent player name and high score using JSON in `Application.persistentDataPath`.
- Breakout-style game scene with paddle, ball, bricks, score, high score display, and restart/menu flow.

Open `Assets/Scenes/MainMenu.unity` in Unity and press Play.
